# Karenderia Web App

A simple web application that helps manage menu items, orders, and daily sales
for a small local eatery (karenderia).

## Technology Used
- HTML
- CSS
- Bootstrap
- JavaScript
- Git & GitHub

## Features
- Menu display using arrays
- Order creation with total calculation
- Daily sales summary
- No database used

## How to Run
1. Open the project in VS Code
2. Open index.html in a browser
